# Mac Starter Site

給 Mac 新手的快捷鍵、推薦文章與必裝 App 教學網站。

## 功能

- Mac 快捷鍵教學與分類篩選
- 快捷鍵搜尋
- 7 天練習路線
- Mac 教學推薦文章連結
- Mac 新手必裝 App 推薦
- 官方資料來源整理

## 本機啟動

```bash
npm start
```

啟動後開啟：

```text
http://127.0.0.1:4173/
```
